# Source code sub-directories

- `aislens`: This is the `aislens` package.
- `MPAS-Tools`: Modified versions of a subset of `landice` data manipulation scripts from [MPAS-Dev/MPAS-Tools](https://github.com/MPAS-Dev/MPAS-Tools)
- `R`: R scripts used for figures
- `scripts`: Python scripts that take CLI arguments and make use of the `aislens` package.
